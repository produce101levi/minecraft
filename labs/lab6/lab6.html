Verificación de contraseñas

¡Bienvenid@! Aquí podrás hacer la verificación de tu contraseña.
A continuación, tras oprimir el botón "¡Ingresa ahora!" tendrás acceso a la sección de validación de contraseñas. Deberás ingresar tu misma contraseña en los dos campos.
Tu contraseña será considerada válida si los dos campos contienen la misma contraseña.
Además, deberá contener al menos una mayúscula, un carácter especial, una minúsucla y un número.

¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?
Es importante usar JavaScript para este propósito porque gracias a esto, la experiencia de usuario mejora substancialmente. Se evitan, por ejemplo, mensajes de error y recargas innecesarias de página. Entre otros beneficios también podemos encontrar una mayor eficiencia, seguridad, reducción de carga en el servidor, etc.
¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?
Si el usuario desactiva JavaScript en su navegador, puede evadir la seguridad de validaciones. Ésta, sin embargo, es una opción extremadamente complicada ya que muchos sitios requieren JavaScript para correr, y estaría muy limitado el sitio. Otras maneras son modificar el código o enviar datos modificados.
Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?
Si bien la validación de datos con JavaScript no garantiza una protección completa (debe de complementarse con otros elementos), es importante validar los datos antes de procesar o almacenarlos, con el fin de garantizar una mejor experiencia de usuario.
